版本兼容

Version Class Incompatible
2.0.8 ProxyFactory 调整了包名，将ProxyFactory从com.alibaba.dubbo.rpc.proxy包上移到com.alibaba.dubbo.rpc根包，如果有扩展此接口，请相应调整。
2.0.9 ZookeeperRegistry 调整了Zookeeper的存储结构，增加providers和consumers分类存储，便于监控中心单独订阅consumers信息，如果有用Zookeeper注册中心的，需要清空Zookeeper注册中心数据。
2.1.0 RegistryService 修改了register()方法，增加了listener参数，有自行扩展注册中心的，请相应调整。
2.2.0 RegistryService 修改了注册契约，支持基于category参数分类存储路由规则等信息，有自行扩展注册中心的，请相应调整。
2.2.0 MonitorService 将count()改为了collect()，并通过参数URL的协议区分数据，如：count://，trace://，log://等，有用SimpleMonitor，需先升级SimpleMonitor，再升级Provider和Consumer的版本，否则会发送监控数据失败。
2.2.0 CacheFactory 因2.1.0新加入的CacheFactory扩展点，设计契约和其它扩展点不一致，调整了getCache()接口，如果有扩展此接口，请相应调整，如果是继承于AbstractCacheFactory，则兼容。
2.2.0 Validator 因2.1.0新加入的Validator扩展点，设计契约和其它扩展点不一致，调整了validate()接口，如果有扩展此接口，请相应调整。
