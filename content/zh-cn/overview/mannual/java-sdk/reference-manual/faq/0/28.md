### 可能的原因

你执行了以下操作之一：
* 尝试或已经调整了 accesslog 的输出位置

### 排查和解决步骤

请检查应用配置中的 `accesslog.fixed.path=true` 开关是否处于开启状态，如未开启则可忽略；如果当前是开启状态，则请确认是否 acesslog 路径切换的行为是否为可信任的人所执行，以避免可能的安全风险。
