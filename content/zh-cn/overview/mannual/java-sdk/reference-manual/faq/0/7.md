### 可能的原因

1. 一般是 `Class.forName(className)` 执行此方法时，找不到 `className` 当前类。
2. 业务代码上显示排除了当前 `className` 类，导致加载时未找到。

### 排查和解决步骤

1. 检查 `Class.forName(className)` 中，`className` 是否存在。
2. 排查业务代码，有没有使用配置或扫描注解 `exclude` 排除了一些类或包。
