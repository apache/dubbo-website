### 可能的原因
在`CacheableFailbackRegistry.RemovalTask`清空 url 缓存时候出错将会触发清空 URL 缓存出错。

### 排查和解决步骤
该错误为 Dubbo 内部错误，如果您遇到可以在 github 创建 **issues** 并提供错误信息以及复现步骤，我们将协助您解决问题。
