### 可能的原因

1. 客户端配置的协议与服务端配置的协议并不匹配。（如客户端配置的协议是 Dubbo 协议，但服务端只能提供 Rest 协议的服务）
2. 注册中心（或配置中心）不可靠，推送了并不合法的数据。



### 排查和解决步骤

1. 检查提供方和消费方双方的协议配置。
2. 更新注册中心的版本。
