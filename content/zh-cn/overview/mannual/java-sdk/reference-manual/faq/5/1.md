### 可能的原因

1. 配置中心所在的服务器关机或宕机。
2. IP 或者端口号写错。
3. 防火墙错误拦截了配置中心的端口。


### 排查和解决步骤

1. 检查配置中心 IP 以及端口的配置。
2. 检查服务器是否开机，是否正常工作。
3. 检查防火墙等是否放行配置中心所用的端口。
