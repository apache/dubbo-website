 Dubbo is a family of libraries for building browser and gRPC-compatible HTTP APIs. If you're tired of hand-written boilerplate and turned off by massive frameworks, Connect is for you.
 * 轻量
 * 简单 API
 * gRPC 兼容
 * 支持 HTTP 访问
