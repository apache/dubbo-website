## 安装

你可以使用 Dubbo 社区提供的示例配置快速安装 Zipkin。

```bash
kubectl create -f https://raw.githubusercontent.com/apache/dubbo-admin/refactor-with-go/deploy/kubernetes/zipkin.yaml
```
> 本安装仅适用于测试或体验使用，生产级别的安装请参考 Zipkin 官方安装文档。
