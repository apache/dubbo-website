Why Dubbo?


Apache Dubbo 是一款支持多语言的、易用的 web 和 rpc 框架，同时为构建企业级微服务提供服务发现、流量治理、可观测、认证鉴权等能力、工具与最佳实践。

{{% /blocks/lead %}} {{< blocks/section id="oceanNodes" >}} {{% blocks/feature image="flower" %}} #### 快速上手，让开发者专注微服务业务开发 多语言 SDK 以贴合各自语言特性的方式定义了服务配置、API 与编程范式，通过丰富的生态组件、配套工具等，轻松解决微服务 day0 到 day1 的问题，让开发者专注在微服务业务开发上。 {{% /blocks/feature %}} {{% blocks/feature image="scalable" icon="fas fa-share-alt" %}} #### 服务治理，随时监测和管控微服务集群状态 提供服务发现、动态配置等基础微服务套件，支持导出多维度的可观测指标并与接入可视化系统如 Dubbo Admin，通过自定义及接入开源 Service Mesh 体系提供强大的流量管控能力。 {{% /blocks/feature %}} {{% blocks/feature image="blocks" icon="fa fa-share-alt" title="Mesh 解决方案" %}} #### 超高性能，面向百万实例、QPS 微服务集群设计 以高性能的二进制与 HTTP 通信协议，提供超低延迟的数据传输与高吞吐量 Streaming 通信模型；横向可扩展的、面向百万集群规模调优的地址推送、连接管理、流量调度实现。 {{% /blocks/feature %}} {{% blocks/feature image="blocks" icon="fa fa-share-alt" title="Mesh 解决方案" %}} #### 企业级解决方案，多年标杆企业生产环境检验 用户群体遍布各行各业，典型代表包括阿里巴巴、工商银行、携程、海尔、金蝶等，云厂商如阿里云、腾讯云、华为云等也推出了基于 Dubbo 的托管云产品。值得关注的是 Dubbo3 在阿里巴巴已全面取代 HSF 实现了框架统一。 {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section id="video" background-image="kub_video_banner_homepage" >}}
