监控中心也是一个标准的Dubbo服务，可以通过注册中心发现，也可以直连。

简易注册中心安装

  1. 暴露一个简单监控中心服务到注册中心: (如果是用安装包，不需要自己写这个配置，如果是自己实现监控中心，则需要)

    ```xml

    ```

  2. 通过注册中心发现监控中心服务:

    xml <dubbo:monitor protocol="registry" />

    或

    dubbo.properties

    xml dubbo.monitor.protocol=registry

  3. 暴露一个简单监控中心服务，但不注册到注册中心: (如果是用安装包，不需要自己写这个配置，如果是自己实现监控中心，则需要)

    ```xml


    ```

  4. 直连监控中心服务

    xml <dubbo:monitor address="dubbo://127.0.0.1:7070/org.apache.dubbo.monitor.MonitorService" />

    或：

    sh <dubbo:monitor address="127.0.0.1:7070" />

    或：

    dubbo.properties

    sh dubbo.monitor.address=127.0.0.1:7070
