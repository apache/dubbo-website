Store DEMO
We provide a fully functional e-commerce system developed using Dubbo. The e-commerce system consists of 11 microservice applications, communicating via the triple RPC protocol. Users can access the store through the frontend web service to browse, use the shopping cart, and place orders.
You can visit demo.dubbo.apache.org to experience the store we've deployed online. If you'd like to learn more about the source code and implementation details, you can visit GitHub to view the source code. We provide implementations in both Java and Go.
