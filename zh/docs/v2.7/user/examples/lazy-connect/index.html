延迟连接

在 Dubbo 中配置延迟连接

延迟连接用于减少长连接数。当有调用发起时，再创建长连接。

<dubbo:protocol name="dubbo" lazy="true" />

提示

该配置只对使用长连接的 dubbo 协议生效。
